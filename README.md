## Started

Clone the repo and make it yours:

```bash
git clone --depth 1 https://github.com/Linusar/react-redux-tdd-todolist
cd react-redux-tdd-todolist
rm -rf .git
```

## Install

```sh
yarn install
```

## Usage

```sh
yarn start
```

## Build

```sh
yarn build
```

## Run unit tests

```sh
yarn test
```

## Run e2e tests

```sh
yarn e2e
```
